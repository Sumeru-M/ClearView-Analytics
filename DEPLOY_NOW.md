## 🎯 DEPLOY IN 3 STEPS

### STEP 1: Go to Render
```
https://render.com/dashboard
```

### STEP 2: Create Web Service
- Click **"New +"**
- Select **"Web Service"**
- Choose **Connect GitHub**
- Select repository: **Sumeru-M/ClearView-Analytics**
- Click **"Connect"**

### STEP 3: Configure & Deploy
- **Name**: `clearview-analytics`
- **Environment**: `Python 3`
- Other settings auto-fill from render.yaml
- Set environment variables:
  - `AUTH_SECRET` = (any secure string)
  - `CORS_ORIGINS` = `*`
- Click **"Create Web Service"**
- Wait 2-5 minutes for deployment
